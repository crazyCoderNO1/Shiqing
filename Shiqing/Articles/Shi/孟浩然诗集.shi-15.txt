Class:Shi
Book:孟浩然诗集
//
Title:留别王维
寂寂竟何待？
朝朝空自归。
欲寻芳草去，
惜与故人违。
当路谁相假？
知音世所稀。
只应守寂寞，
还掩故园扉。  
//
Title:清明日宴梅道士房
林卧愁春尽，
开轩览物华。
忽逢青鸟使，
邀入赤松家。
丹灶初开火，
仙桃正发花。　
童颜若可驻，
何惜醉流霞！
//
Title:题义公禅房     
义公习禅寂，
结宇依空林。
户外一峰秀，
阶前众壑深。
夕阳连雨足， 
空翠落庭阴。  
看取莲花净， 
方知不染心。
//
Title:临洞庭         
八月湖水平， 
涵虚混太清。
气蒸云梦泽，
波撼岳阳城。
欲济无舟楫，
端居耻圣明。
坐观垂钓者，
徒有羡鱼情。   
//
Title:宿建德江
移舟泊烟渚，
日暮客愁新。
野旷天低树，
江清月近人。
//
Title:春晓
春眠不觉晓，
处处闻啼鸟。
夜来风雨声，
花落知多少。    
//
Title:夏日南亭怀辛大 
山光忽西落，
池月渐东上。
散发乘夕凉，
开轩卧闲敞。
荷风送香气，
竹露滴清响。
欲取鸣琴弹，
恨无知音赏。
感此怀故人，
中宵劳梦想。
//
Title:秋登兰山寄张五
北山白云里，
隐者自怡悦。
相望始登高，
心随雁飞灭。
愁因薄暮起，
兴是清秋发。
时见归村人，
沙行渡头歇。
天边树若荠，
江畔洲如月。
何当载酒来，
共醉重阳节。
//
Title:宿业师山房待丁大不至
夕阳度西岭，
群壑倏已暝。
松月生夜凉，
风泉满清听。
樵人归欲尽，
烟鸟栖初定。
之子期宿来，
孤琴候萝径。
//
Title:夜归鹿门山歌
山寺钟鸣昼已昏，
渔梁渡头争渡喧。
人随沙路向江村，
余亦乘舟归鹿门。
鹿门月照开烟树，
忽到庞公栖隐处。
岩扉松径长寂寥，
惟有幽人自来去。
//
Title:与诸子登岘山
人事有代谢，
往来成古今。
江山留胜迹，
我辈复登临。
水落鱼梁浅，
天寒梦泽深。
羊公碑字在，
读罢泪沾襟。
//
Title:早寒江上有怀
木落雁南渡，
北风江上寒。
我家襄水曲，
遥隔楚云端。
乡泪客中尽，
孤帆天际看。
迷津欲有问，
平海夕漫漫。
//
Title:岁暮归南山
北阙休上书，
南山归敝庐。
不才明主弃，
多病故人疏。
白发催年老，
青阳逼岁除。
永怀愁不寐，
松月夜窗墟。
//
Title:过故人庄
故人具鸡黍，
邀我至田家。
绿树村边合，
青山郭外斜。
开轩面场圃，
把酒话桑麻。
待到重阳日，
还来就菊花。
//
Title:秦中感秋寄远上人
一丘尝欲卧，
三径苦无资。
北土非吾愿，
东林怀我师。
黄金燃桂尽，
壮志逐年衰。
日夕凉风至，
闻蝉但益悲。 
//
Title:宿桐庐江寄广陵旧游
山暝听猿愁，
沧江急夜流。
风鸣两岸叶，
月照一孤舟。
建德非吾土，
维扬忆旧游。
还将两行泪，
遥寄海西头。
//
Title:访袁拾遗不遇 
洛阳访才子，
江岭作流人。
闻说梅花早，
何如此地春。
